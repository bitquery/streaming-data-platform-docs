## Jupiter Swaps & Trading Data

### Track Latest Swaps on Jupiter

Get real-time Jupiter aggregator swap data including tokens involved, routing paths, and account details.

We monitor Jupiter's program address `JUP6LkbZbjS1jKKwapdHNy74zcZ3tLUZoi5QNyVTaV4` for `sharedAccountsRoute` instructions to track swap activity.
The query returns tokens involved in swaps, source and destination addresses, and routing information.

[Jupiter Latest Swaps — Stream ➤](https://ide.bitquery.io/Tokens-involved-in-Jupiter-swap-source-address-destination-address-DEX-involved_2#)

<details>
  <summary>Click to expand GraphQL query</summary>

```graphql
subscription {
  Solana {
    Instructions(
      where: {
        Instruction: {
          Program: {
            Address: { is: "JUP6LkbZbjS1jKKwapdHNy74zcZ3tLUZoi5QNyVTaV4" }
            Method: { is: "sharedAccountsRoute" }
          }
        }
        Transaction: { Result: { Success: true } }
      }
    ) {
      Transaction {
        Signature
      }
      Instruction {
        Program {
          Method
          AccountNames
          Address
        }
        Accounts {
          Address
          IsWritable
          Token {
            Mint
            Owner
            ProgramId
          }
        }
      }
    }
  }
}
```

</details>
