# Bitquery 中文文档

Bitquery 的完整文档目前以**英文**提供。您可以使用浏览器的自动翻译功能，用中文阅读所有页面。

## 使用自动翻译

- **Chrome / Edge**：在任意文档页面右键单击 → 选择“翻译成中文”。
- **Safari**：点击地址栏中的翻译图标 → 选择中文。

翻译后即可浏览全部内容，包括代码示例。
